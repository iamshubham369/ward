# 🛡️ WardPulse OS: Tactical Civic Intelligence

**WardPulse OS** is a high-fidelity, military-grade administrative platform designed for modern municipal ward governance. It bridges the gap between citizens and authorities through real-time telemetry, AI-driven performance diagnostics, and cinematic geospatial visualization.

## 🚀 Key Modules & Intelligence Layers

### 1. 🗺️ Cinematic Tactical Mapping
*   **Indian View Initializer:** Starts with a global view of India and executes a smooth cinematic fly-in to the specific ward.
*   **3D Satellite Terrain:** Toggle between "Cyber Dark" and high-resolution "Google Hybrid" satellite imagery for precise infrastructure monitoring.
*   **Neon Node Tracking:** Glowing, pulsating radar dots represent hazards (Red) and ongoing projects (Blue).

### 2. 📊 AI-Driven Performance Engine
*   **Governance Analytics:** Real-time scoring of administrative efficiency.
*   **Sector Diagnostics:** Detailed tracking of Roads, Water, Electricity, and Sanitation sectors.
*   **Sentiment Telemetry:** Analyzing citizen buzz and community demand.

### 3. 📄 Official Mandate Generation
*   **Pro PDF Engine:** Generates high-end, printable A4 administrative reports with dynamic grade badges, progress bars, and institutional branding.
*   **Infrastructure Ledger:** Automatically compiles project data into professional ledgers for municipal audits.

### 4. 🕵️ Global Overlord Console (Admin)
*   **Easter Egg Access:** Hidden from the public. Click the **WardPulse OS Logo 5 times** to unlock.
*   **Master Control:** Manage all provisioned ward nodes and identity registries from a single command center.

## ⚙️ Tactical Deployment Instructions

### 1. Prerequisites
*   Node.js (v18+)
*   NPM or Yarn

### 2. Setup Backend
```bash
cd backend
npm install
npm run dev
```
*Note: Backend defaults to port 5000.*

### 3. Setup Frontend
```bash
cd frontend
npm install
npm run dev
```
*Note: Portal accessible at http://localhost:5173.*

### 4. Quick Start (Windows)
Run the `start.bat` file in the root directory to launch both systems simultaneously.
